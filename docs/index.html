Framework

This folder contains the framework code for the family apps suite. Includes:

  • Federated Modules Reusable front end components
  • React Components Reusable React components
  • Shared Utilities Reusable utilities

Contents

./modules - Home for all federated modules. Create new reusable components here and expose them in the components folder to be used remotely. ./webpack-react-mf-app2 - The app that is used to test the framework. This is where new federated modules can be imported remotely and tested.

Use the setup-mf-app.sh script to automate the setup of a new Federated Module app that is preconfigured to use the framework.

Framework is opinionated and is not intended for general use. Dependencies:

  • React
  • Webpack
  • React-Query
  • React-Forms
  • MUI optional
  • ShadCdn optional
    • Tailwind CSS optional
